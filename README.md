# Instructions for this documentation

## Running Locally

For more instructions, please visit the [Docsify website](https://docsify.js.org)

### Install Docsify-cli

```
npm i docsify-cli -g
```

### Serve the file

```
docsify serve docs
```

It should run locally on `http://localhost:3000`.
